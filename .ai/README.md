# README

## FAQs

### How to implement large modules?

Divide your module in multiple phase and start with MVP of module in docs layer and let AI finish that. Once done, Update module docs again for next phase and let AI work on that. Repeat this process until you have your complete module.

### HITL is consuming too much context at end, how to improve?

We can try ralph where we first define prd.json & progress.txt based on our module docs and once it's ready spin up ralph and let it work on that.
