Complete setup guide to get LaunchDayOne running on your machine.

## Prerequisites

- Node.js 22.x or LTS
- pnpm 10.22.0+
- Git
- Docker (optional, for local database)

## Installation

::steps{level="3"}

### Clone Repository

```bash
git clone https://github.com/LaunchDayOne/LaunchDayOne.git
cd LaunchDayOne
```

### Configure Environment

1. Copy `.env.example` to `.env`:

    ```bash
    cp .env.example .env
    ```

2. Open `.env` and configure variables. Refer to following guides for easier local setup:

    :::accordion

    ::accordion-item{label="Using docker based Postgres" icon="i-lucide-database"}

    ```bash
    docker run --name <your_project_postgres> \
      -e POSTGRES_USER=<pg_user> \
      -e POSTGRES_PASSWORD=<your_pwd> \
      -e POSTGRES_DB=<your_project> \
      -p 5432:5432 \
      -v <your_project_pg_data>:/var/lib/postgresql/data \
      -d postgres:17.5-alpine
    ```

    ::
    :::

    :::accordion

    ::accordion-item{label="Use MailPit to test emails locally" icon="i-lucide-database"}

    Refer to official docs of [MailPit](https://github.com/axllent/mailpit) to run it in docker.

    ::
    :::

### Install Dependencies

```bash
pnpm install
```

### Push schema to database

```bash
pnpm db:push
```

### Start Development Server

```bash
pnpm dev

# Most likely, You can access at http://localhost:3000
```

::
